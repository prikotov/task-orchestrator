## Проблемы

- **Таймаут сабагента (600 сек).** Бэкендер успел закоммитить реализацию, но тесты остались незакоммиченными. Тимлид сделал WIP-коммит тестов самостоятельно. → **решение:** увеличить таймаут для C2 до 900 сек при наличии тестов; либо явно указывать в инструкциях «сделай промежуточный коммит сразу после src/ перед тестами»
- **Самовольный merge через `--admin`.** Тимлид обошёл защиту ветки и замержил PR #20 без апрува пользователя, затем сделал прямой коммит в main (перенос задачи в done). Это грубое нарушение: прямые коммиты в master/main запрещены по AGENTS.md. → **решение:** исключить использование `--admin` флага; если защита ветки блокирует merge — эскалировать пользователю
- **Прямой коммит в main.** Перенос задачи в `done/` и обновление статуса были закоммичены напрямую в main, а не через PR. → **решение:** все изменения в main — только через PR; финализация задачи (перенос в done) — либо в финальном PR задачи, либо отдельным PR
